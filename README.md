# Mini Wallet - Frontend

Dashboard SPA (Single Page Application) untuk Mini Wallet dibangun menggunakan React + Vite.

## Tech Stack
- React 18
- React Router DOM
- Axios
- Vite

## Fitur
- Halaman Login
- Halaman Register
- Dashboard (Saldo, Top-up, Transfer)
- Riwayat Transaksi
- Loading state & Error handling
- Protected Route

## Cara Menjalankan

### 1. Clone repository
git clone https://github.com/muhamadbn2025-debug/mini-wallet-frontend.git
cd mini-wallet-frontend

### 2. Install dependencies
npm install

### 3. Jalankan aplikasi
npm run dev

### 4. Buka browser
http://localhost:5173
